# Waifu Roulette Lounge - Specification

## Project Overview
- **Name:** Waifu Roulette Lounge
- **Type:** Web Application (Vite + React)
- **Purpose:** Anime-themed roulette lounge entertainment demo
- **Target:** Entertainment/fun demo with fake chips

## ⚠️ Disclaimer

**This project uses simulated currency only. No real money wagering or payouts.**

## Features

- Anime-themed roulette game
- Fake chip betting system
- Animated dealer reactions
- Local-only state (no backend)

## Tech Stack
- Vite + React
- Tailwind CSS
- Lucide React icons
