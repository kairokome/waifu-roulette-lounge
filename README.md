# Waifu Roulette Lounge

Anime-themed roulette lounge with fake chips, animated dealer reactions, and local-only persistence. Entertainment only.

## ⚠️ Disclaimer

**This project uses simulated currency only. No real money wagering or payouts.**

This is purely entertainment software. All chips and winnings are virtual/fake and have no real-world value.

## Features

- Anime-themed roulette game
- Animated dealer reactions
- Fake chip betting system
- Local-only persistence (no server)
- Entertainment purposes only

## Tech Stack

- Vite + React
- Tailwind CSS
- Lucide React icons

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
